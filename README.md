# Financial Data Analysis Dashboard (Excel Project)

##  Project Description

The **Financial Data Analysis Dashboard** is an Excel-based project designed to analyze and visualize key financial metrics such as **sales, profit, cost of goods sold (COGS), discounts, and units sold**.

This project transforms raw financial data into an **interactive and visually appealing dashboard**, enabling users to understand business performance, identify trends, and make data-driven decisions.

---

##  Objectives

* Analyze financial data to evaluate business performance
* Identify trends in sales, profit, and cost
* Compare performance across products, regions, and segments
* Measure the impact of discounts on profitability
* Build an interactive dashboard using Excel

---

##  Dataset Overview

The dataset includes the following key attributes:

* `Product` – Product name
* `Country` – Region of sales
* `Segment` – Business segment (Government, Enterprise, etc.)
* `Sales` – Total revenue generated
* `Profit` – Net profit earned
* `COGS` – Cost of Goods Sold
* `Discount` – Discount applied
* `Units Sold` – Number of units sold
* `Date` – Month and year
